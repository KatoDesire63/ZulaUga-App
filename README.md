[ZulaUga-App](https://katodesire63.github.io/ZulaUga-App/) by Kato Desire.

This app showcases all the beautiful sceneries of Uganda. For example torist attractions, local cuisines, and cultural 
heritage of Uganda. It also enables both local and international tourists to book for accommodations, submit testimonials,
use a map to find nearby accommodations and tourist attractions.
And also make payments for seamless user exprience.

##Features
- **Home Page**: Overview of tourist attractions with a call to action to book a trip.
- **Attractions Page**: Detailed descriptions of various tourist attractions, categorized for easy navigation.
- **Gallery Page**: Photo gallery of beautiful tourist spots.
- **Booking Page**: Book a trip with payment integration for Mobile Money (Airtel and MTN for local tourists),Visa Card,and PayPal.
- **Testimonials**: User testimonials with backend intergration to fetch and display dynamically.
- **Map Page**: Interactive map showing tourist attractions and  nearby accommodations using OpenStreetMap and Leaflet.
- **Cuisines Page**: Showcases Ugandan local cuisines with attractive themes, colors, and animations.

  ## Technologies Used

  - **Frontend**: HTML, CSS, JavaScript.
  - **Backend**: Node.js, Express.js, MongoDB.
  - **APIs**: MetaWeather for weather information, OpenStreetMap with Leaflet for maps.
  - **Payment Intergration**: Mobile Money(Airtel and MTN), Visa Card, PayPal.
